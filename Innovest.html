Innovest – Microinvestment Platform

Making smart investments accessible for every Indian citizen

Problem Statement

In India, many—especially women, students, and first-time earners—lack awareness about investment schemes. Existing platforms are often complex or generic, leading to underutilization of opportunities. Innovest bridges this gap with personalized, accessible guidance.

Objective

To build Innovest, a personalized, interactive web-based investment advisory platform that:

  • Educates users about schemes tailored to their demographics
  • Provides chatbot-driven financial advice
  • Recommends schemes based on age, gender, and budget

Key Features

  • Interactive homepage UI with clean navigation
  • User login and profile-based personalization
  • Smart chatbot advisor simplifying investment decisions
  • Dynamic scheme suggestions using user data
  • Detailed scheme pages categorized by need
  • Seasonal animated UI (snow, trees, house)
  • FAQ and feedback system for improvement

Tech Stack

  • Frontend: HTML5, CSS3, JavaScript, Responsive Design
  • Chatbot & NLP: Dialogflow
  • Backend: Node.js + Express.js
  • Database: MongoDB
  • Animations & UI: CSS3, custom illustrations
